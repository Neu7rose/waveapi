GET /search

Busca músicas na base Wave API por nome, artista ou combinação. Retorna até 10 resultados por padrão. Suporta paginação.

Parâmetro Tipo Obrigatório Descrição
q string Obrigatório Termo de busca (nome ou artista)
limit integer Opcional Máx. de resultados (padrão: 10)
page integer Opcional Número da página (padrão: 1)

Retorna os dados completos de uma música específica pelo seu identificador único.

ParâmetroTipoObrigatórioDescrição

Campos especiais MTA:SA

url_with_grave
Link encapsulado com acento grave (`). Compatível com playSound() no MTA:SA.
url_clean
URL pura sem caracteres especiais. Ideal para armazenamento em banco de dados ou exibição na interface.

AUTH Autenticação

A Wave API usa autenticação via Bearer Token. Inclua o header em todas as requisições autenticadas. No plano Free, algumas rotas funcionam sem autenticação com limite reduzido.

Obtenha seu token em waveapi.fun/dashboard após criar sua conta.

1.5.8+

music_system.lua LUA
-- Wave API no MTA:SA
-- Busca e toca música via HTTP

local WaveAPI = {}
WaveAPI.token = "SEU_TOKEN"
WaveAPI.base = "https://waveapi.fun"

function WaveAPI.search(query, cb)
  local url = WaveAPI.base
    .. "/search?q=" .. query
  fetchRemote(url, {
    headers = { Authorization =
      "Bearer " .. WaveAPI.token }
  }, function(data)
    local song = fromJSON(data)
    -- url_with_grave já tem os `
    local sound = playSound(
      song.results[1].url_with_grave
    )
    cb(sound)
  end)
end

-- Uso:
WaveAPI.search("Eminem", function(s)
  setSoundVolume(s, 0.8)
end)

FAQ

Perguntas frequentes

O que é o acento grave e por que ele é necessário no MTA:SA?
+
No MTA:SA, a função playSound() requer que URLs de streams externos sejam passadas com um acento grave (`) no início e no fim. O campo url_with_grave da Wave API já entrega o link formatado corretamente, eliminando a necessidade de manipulação manual da string em Lua.
Preciso de autenticação para usar a API?
+
No plano Free, você pode fazer até 500 requisições por dia sem nenhum token. Para volumes maiores ou acesso a recursos avançados, você precisará de um token Bearer obtido no painel após criar uma conta.
A Wave API funciona com outros jogos além do MTA:SA?
+
Sim! Embora tenha sido otimizada para MTA:SA, qualquer aplicação pode usar a Wave API. O campo url_clean retorna a URL sem formatação especial, compatível com qualquer linguagem e plataforma.
Qual é o limite de requisições por segundo?
+
O plano Free permite até 10 requisições por segundo. O plano Pro eleva esse limite para 50 req/s. Em caso de excesso, a API retorna status 429 (Too Many Requests) com o header Retry-After indicando quando tentar novamente.
Como faço para adicionar músicas à base de dados?
+
Atualmente a curadoria é feita pela nossa equipe para garantir qualidade e disponibilidade dos links. Se precisar de uma música específica, abra uma solicitação no Discord e avaliaremos em até 48 horas. Usuários Pro têm prioridade nas solicitações.
Os links expiram?
+
Os IDs das músicas são permanentes — a URL /song/{id} sempre retorna os dados atualizados. Recomendamos não armazenar os links diretamente pois podem ser atualizados; sempre busque via ID para garantir disponibilidade.
